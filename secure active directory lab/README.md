Secure Active Directory Home Lab

Project Overview

This project demonstrates the deployment and configuration of a secure Active Directory environment using Windows Server 2022 in a virtualized lab environment. The system simulates a small enterprise network by configuring a Domain Controller, creating user accounts, and applying Group Policy security settings.
The project was completed as part of independent learning and practical skill development in system administration and cybersecurity fundamentals.

Objectives

•	Deploy a Windows Server Domain Controller

•	Configure Active Directory Domain Services (AD DS)

•	Create and manage user accounts

•	Implement Group Policy security controls

•	Simulate a secure enterprise network environment

Technologies Used

•	Windows Server 2022

•	Oracle VirtualBox

•	Active Directory Domain Services (AD DS)

•	Group Policy Management Console (GPMC)

•	Windows Server Manager

Lab Environment Configuration

•	4 GB RAM allocated to virtual machine

•	2 CPU cores assigned

•	50 GB virtual hard disk

•	Network Adapter 1 configured as NAT

•	Network Adapter 2 configured as Internal Network

•	Domain name configured as:
lab.local

Key Tasks Performed

1.	Installed Windows Server 2022 on a virtual machine
2.	Configured a static IP address
3.	Installed Active Directory Domain Services
4.	Promoted the server to a Domain Controller
5.	Created user accounts
6.	Configured Group Policy security settings
7.	Tested system functionality and security policies
