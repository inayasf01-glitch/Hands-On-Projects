Post-Attack Digital Forensic Investigation Lab

Project Overview

This project demonstrates the process of conducting a digital forensic investigation on a compromised system using professional forensic tools within a virtualized lab environment. The investigation simulates a post-attack scenario where evidence from a previously exploited system is captured, analyzed, and documented to determine the sequence of events during a security breach. The project was completed as part of independent learning and practical skill development in computer forensics and incident response.

Objectives

• Capture and preserve digital evidence from a compromised virtual machine

• Analyze forensic images using Autopsy to extract artifacts

• Recover deleted files, browser history, and system logs

• Generate a timeline of attacker activity within the system

• Document findings and maintain chain-of-custody procedures

• Demonstrate professional incident response methodology

Technologies Used

• FTK Imager (Forensic Imaging Tool)

• Autopsy (Digital Forensics Platform)

• Oracle VirtualBox

• Ubuntu / Windows Operating System (Compromised VM)

• Kali Linux (Optional for attack replication)

• Draw.io (Network and Case Visualization)

Lab Environment Configuration

• 4 GB RAM allocated to virtual machine

• 2 CPU cores assigned

• 50 GB virtual hard disk

• Network Adapter 1 configured as NAT

• Network Adapter 2 configured as Internal Network

• Compromised VM configured as target system for investigation

Key Tasks Performed

1.	Created a forensic image of the compromised virtual machine using FTK Imager
2.	Launched Autopsy and created a new case to import the forensic image
3.	Ran Autopsy ingest modules to extract digital artifacts including browser history, recently executed programs, and deleted files
4.	Conducted timeline analysis to document attacker activity and sequence of events
5.	Verified hash values of evidence files to ensure integrity
6.	Generated a professional forensic report summarizing findings and recommended remediation

Skills Demonstrated

• Digital evidence acquisition and preservation

• Forensic imaging using FTK Imager

• Artifact extraction and analysis using Autopsy

• Timeline reconstruction of security incidents

• Chain-of-custody maintenance

• Professional forensic reporting

• Incident response procedures

• Virtual lab setup and secure evidence handling
